# 🚀 Presentación de Mi Portafolio Web Interactivo & AI Assistant

¡Hola a todos en mi red! 👋 

Hoy quiero compartir con ustedes un hito muy importante en mi desarrollo profesional. He diseñado, desarrollado y desplegado desde cero mi **Portafolio Web Interactivo**, operando como una Single Page Application (SPA) completa.

Más que un simple currículum digital, es un ecosistema completo diseñado para demostrar mis capacidades técnicas y mi enfoque hacia la resolución de problemas reales integrando Inteligencia Artificial.

## 🛠️ Stack Tecnológico (La Arquitectura)

Para construir un MVP robusto y escalable, seleccioné herramientas modernas orientadas al rendimiento y la seguridad:

*   **Frontend:** React 18, TypeScript, Vite y Tailwind CSS para una interfaz fluida, fuertemente tipada y 100% responsiva.
*   **Backend & Cloud:** Arquitectura Serverless utilizando Firebase (Firestore para base de datos en tiempo real, Storage para gestión multimedia, y Auth para seguridad y control de acceso).
*   **Inteligencia Artificial:** Integración nativa con la API de **Google Gemini Pro Vision**.
*   **Infraestructura:** Despliegue continuo (CI/CD) de alta velocidad.

## ✨ Características Principales

### 1. Panel de Administración Personalizado (CMS Propio)
Desarrollé un panel de control privado y seguro que me permite gestionar todo el contenido de la web de forma dinámica, sin necesidad de tocar código.
*   **Editor de Texto Enriquecido:** Integración de *ReactQuill* para redactar los detalles de los proyectos y publicar artículos completos en el Blog directamente desde la web, con formateo HTML nativo.
*   **Gestión de Almacenamiento:** Subida de imágenes (portadas y galerías de proyectos) automatizada hacia Firebase Storage bajo estrictas reglas de seguridad CORS.

### 2. AndyBot: Mi Asistente Virtual Inteligente 🤖
La joya de la corona del portafolio. He implementado un asistente virtual (AndyBot) accesible desde cualquier página de la web, impulsado por Gemini AI, con un modo secreto muy especial:

*   **Modo Estratega de Carrera:** Mediante un sistema seguro de "Santo y Seña", el bot asume el rol de mi *Estratega de Carrera*.
*   **Análisis Multimodal:** Es capaz de procesar capturas de pantalla de vacantes (Job Descriptions) que el usuario pegue en el chat.
*   **Hibridación y Generación de PDF en Tiempo Real:** El bot cruza los requerimientos de la vacante analizada con mis métricas y experiencia *hardcodeada* (sin inventar datos). Retorna una estructura JSON que el motor frontend (jsPDF) intercepta e hidrata automáticamente para descargar en el navegador del reclutador un **Curriculum Vitae en PDF 100% adaptado a esa vacante específica** en cuestión de segundos.
